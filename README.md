## **📦 Project Setup (Vue 3 + Vite + JavaScript)**

### **🚀 프로젝트 시작하기**

**1. 레포지토리 클론**

```
git clone https://github.com/MyHomeCatch/MyHomeCatch_Front.git
cd MyHomeCatch_Front
```

**2. 패키지 설치**

```
npm install
```


**3. 개발 서버 실행**

```
npm run dev
```

브라우저에서 [`http://localhost:5173`](http://localhost:5173/)로 접속하여 프론트 개발 서버를 확인할 수 있습니다.
